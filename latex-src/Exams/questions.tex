\section*{Useful Identities}

For $L^p$ spaces we define the conjugate function $f*$ by 
$$f^* = \|f\|_p^{1-p}\text{sign}(f)|f|^{p-1}$$
Then we have 
$$\|f\| = \int_E ff^*$$
